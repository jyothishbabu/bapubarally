III. FAQ

1) How can I activate Slider?

Full screen slider is activated in the wordpress admin on page "Appearance > Theme Options > Customize Your Site > Header area > Header type".
Once there under "Front page header type" select "Full screen slider" and click on button "Save & Publish".

2) Does this theme works with WooCommerce?

Unfortunately WooCommerce is supported only in the PRO theme version. 
By purchasing PRO theme you will also get Theme Support for One Year. 
You can purchase PRO theme on this page: http://dessky.com/theme/very-simple-start-pro .
